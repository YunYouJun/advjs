# 术语表

> 供 AI 创作时参考，确保世界观一致性。

| 术语     | 定义                         | 备注                             |
| -------- | ---------------------------- | -------------------------------- |
| 星之记忆 | 每个角色被封印的关键记忆     | 不要用「星的记忆」或「星辰记忆」 |
| 星见市   | 故事发生的主要城市           | 虚构城市，靠海的中型城市         |
| 解锁     | 触发条件使封印记忆显现的过程 | 不是「觉醒」也不是「释放」       |
| 闪回     | 记忆片段短暂浮现但未完全解锁 | 通常伴随画面变白的视觉效果       |
| 天文社   | 学校社团，观星和天文研究     | 凯可能加入的社团之一             |
| 文学社   | 学校社团，文学创作和讨论     | 凯可能加入的社团之一             |
