# 星见神社

## 描述

星见市内历史最悠久的神社，坐落在小山丘上。
据传说这里是「星之记忆」传说的起源地。
每年夏日祭是星见市最热闹的时候。

## 氛围

- **日常**：幽静、庄严，石阶两旁有古老的杉树
- **夏日祭**：热闹、温暖，灯笼和摊位营造出梦幻氛围
- **深夜**：神秘、静谧，据说能看到特别的星空
