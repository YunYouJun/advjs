# 学校天台

## 描述

位于教学楼顶层的开放天台，被铁丝网围栏环绕。
傍晚时分可以看到远处的海面在夕阳下闪烁。
是艾莉亚最喜欢的秘密场所。

## 氛围

- **白天**：安静、空旷，偶尔有风声
- **傍晚**：温暖、感伤，夕阳余晖
- **夜晚**：清冷、浪漫，可以看星星
