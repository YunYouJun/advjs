## 可能性

- [ChatGPT](https://chat.openai.com/): 基于GPT-3.5架构的大型语言模型，也许你可以通过与它对话进行文本的辅助创作。
- [NovelAI](https://novelai.net/): 有辅助故事写作以及文本到图像生成功能，你可以通过关键词随机生成一些想要的图形，也许这将会是一个很好的素材来源。
