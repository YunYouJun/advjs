# 路线图

ADV.JS Skills 体系的未来发展方向。

## 当前版本 (v0.1)

- ✅ `adv-story` Skill — 基础互动叙事播放
- ✅ CLI `adv play` 命令组
- ✅ JSON 输出格式支持
- ✅ 会话持久化

## 近期计划 (v0.2)

### 增强 adv-story

- [ ] 支持多章节故事（跨 flow node 导航）
- [ ] 存档/读档功能（save/load game state）
- [ ] 角色立绘状态 ASCII 描述
- [ ] 背景音乐提示

### 新 Skills

- [ ] `adv-editor` — AI 辅助剧本编写
  - 生成 `.adv.md` 剧本
  - 语法检查和优化建议
  - 自动生成分支结构

- [ ] `adv-debug` — 游戏调试助手
  - 自动遍历所有分支路径
  - 检测死路径和无限循环
  - 生成测试覆盖报告

## 长期目标

### Agent 生态

- [ ] MCP Server 集成（替代 CLI exec 方式）
- [ ] 多 Agent 协作叙事（多角色扮演）
- [ ] 实时观众互动（投票选择分支）

### 创作工具

- [ ] AI 辅助角色设计
- [ ] 自动场景配图生成
- [ ] 音效/BGM 智能推荐

### 平台支持

- [ ] Web Widget 嵌入
- [ ] Discord Bot 集成
- [ ] 微信小程序集成
